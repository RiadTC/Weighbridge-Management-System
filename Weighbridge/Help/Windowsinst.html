How to install on Windows

WMS application requires a properly running MySQL server (or Mariadb). Since WMS is a 32bit application, you need to install a 32bit version of MySQL or a 32bit version of the Wampserver application. WMS can run on Windows XP. The following steps are required prior to running WMS.

1) Decide on the database name, user name, and password for WMS to use. The database should be unique to WMS.

2) Create the user that will manage the WMS database. For Example, WMSadmin@localhost.

Open a terminal and go to the bin directory of the MySQL installation, then enter the following commands:

mysql -u root -p

Enter password:

CREATE DATABASE Weighbridge;

CREATE USER WMSadmin@localhost;

GRANT ALL PRIVILEGES ON Weighbridge.* TO WMSadmin@localhost IDENTIFIED BY 'password';

FLUSH PRIVILEGES;

exit

An application like phpMyAdmin can be used to complete the above operations.

3) Install and start the system:

Create a new folder in your home directory (for example ‘wms’) and unpack the downloaded compressed file into that folder.

You need to copy the following files from the MySQL directory to the WMS directory: libmysql.dll, mysql.exe, and mysqldump.exe.

Start the application by running the executable file ‘wms.exe’.

4) Once the application is running go to File→Server ,a pop up menu will show. Enter the following info:

  • Database: enter the name of the database that will store your data, the same you entered in step 2, in our example ‘Weighbridge’.

  • Host: server computer name, for example ‘localhost’.

  • Port: 3306 unless you have changed default port number for MySQL

  • User Name: the name you entered in step 2, in our example ‘WMSadmin’.

  • Password: The same password you entered in step 2.

5) Now go to File→Restore and choose the file ‘wmsinit.sql’. The application will initialize the database for use.

6) Goto to File→Login. If all is well you should be able to login using ‘Admin’ for operator and ‘admin’ for password.

How to Install

1. In windows 7/8, install the application using 'run as administrator' option' while executing

setup.exe file.

2. Double-click on setup.exe to install the application. It is a 137-KB file. If you cannot see the

extension, you can enable file extensions using folder options.

3. Install the application. A link will be created in start menu. Copy the link to the desktop if

required.

4. In windows 7/8, it is recommended to set the 'run as administrator' option for smartw.exe file in

the installation folder.

5. Execute the application by clicking on the link in the start-menu. Log in using 'trial' as

password.

6. Mail us at support@smartbglobal.com if you need any help.

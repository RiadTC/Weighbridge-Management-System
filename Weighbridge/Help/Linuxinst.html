How to install on Linux

WMS will run on most Linux distributions. It has been tested on Fedora, Linux Mint, Ubuntu, and Suse Leap 42.x.

WMS application requires a properly running MySQL server (or Mariadb). The following steps are required prior to running WMS.

1) Decide on the database name, user name, and password for WMS to use. The database should be unique to WMS.

2) Create the user that will manage the WMS database. For Example, WMSadmin@localhost.

Open a terminal and enter the following commands:

mysql -u root -p

Enter password:

CREATE DATABASE Weighbridge;

CREATE USER WMSadmin@localhost;

GRANT ALL PRIVILEGES ON Weighbridge.* TO WMSadmin@localhost IDENTIFIED BY 'password';

FLUSH PRIVILEGES;

exit

An application like phpMyAdmin can be used to do the above operations.

3) Install and start the system:

  • Create a new folder in your home directory (for example ‘wms’) and unpack the downloaded compressed file into that folder

  • Set a link to the executable file ‘wms.AppImage’ to run it from the applications menu or just start it by issuing the command ‘~/wms/wms.AppImage’ from the console.

4) Once the application is running go to File→Server ,a pop up menu will show. Enter the following info:

  1. Database: enter the name of the database that will store your data, the same you entered in step 2, in our example ‘Weighbridge’.

  2. Host: server computer name, for example ‘localhost’.

  3. Port: 3306 unless you have changed default port number for MySQL

  4. User Name: the name you entered in step 2, in our example ‘WMSadmin’.

  5. Password: The same password you entered in step 2.

5) Now go to File→Restore and choose the file ‘wmsinit.sql’. The application will initialize the database for use.

6) Goto to File→Login. If all is well you should be able to login using ‘Admin’ for operator and ‘admin’ for password.
